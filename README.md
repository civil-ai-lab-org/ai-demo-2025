# AIデモノートブック集

研究室紹介で使ったAIデモのノートブックです。Google Colabで開いて、自由に試してみてください！

## 使い方
1. このリポジトリをzipでダウンロードする。
2. Google Colabにアップロードする。
3. ノートブックを開いて、コードを実行する。

## デモ一覧
- **画像認識**: 画像をアップロードして、AIに認識させよう！
- **画像検出**: 画像内のオブジェクトを検出してみよう！
- **テキスト生成**: AIに文章を続きから書かせよう！
- **画像キャプショニング**: 画像から説明文を生成しよう！
- **画像生成**: テキストから画像を生成しよう！

## 注意
- Google Colabの無料GPUを使う場合、「ランタイム」→「ランタイムのタイプを変更」→「GPU」を選択してください。
- 画像生成はGPUが必要です。
